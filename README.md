# KmeansGui
A PyQt5 program demonstrating the k means clustering algorithm step by step

### Installation
Install numpy, pandas, matplotlib and PyQt5 if you haven't already

```
pip install numpy
pip install pandas
pip install matplotlib
pip install PyQt5
```
